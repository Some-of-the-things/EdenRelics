Security Assessment

Last updated: April 2026

1. Our Commitment

Eden Relics is committed to maintaining the highest standards of security for our customers' personal data, payment information, and all interactions with our platform. We conduct regular security assessments and continuously improve our security practices.

2. Assessment Overview

A comprehensive security assessment was conducted covering our e-commerce platform, payment processing workflows, banking integrations, and personal data handling practices. The assessment combined code-level review with endpoint testing to identify potential risks and verify the effectiveness of existing security controls.

The assessment confirmed that the platform is built on a solid security foundation with industry-standard authentication, encrypted communications, security headers, and rate limiting. A number of improvements were identified and a remediation programme was undertaken to address all findings.

3. Areas Assessed

  • Payment processing security and integrity
  • Banking integration security
  • Personal data protection and GDPR alignment
  • Authentication and session management
  • Input validation and injection prevention
  • Infrastructure and configuration security

4. Remediation Outcomes

All immediate and high-priority findings have been addressed. Key improvements include:

  • Customer data is protected by robust ownership verification
  • Security tokens meet current cryptographic standards
  • Third-party account linking requires proper authorisation
  • Comprehensive input validation is enforced across all user-facing interfaces
  • Cross-origin and browser permission policies have been tightened
  • Error responses do not expose internal system information
  • Rate limiting covers all sensitive operations

Remaining lower-priority items are being addressed progressively across ongoing development as part of our continuous security improvement programme.

5. Ongoing Security Practices

Eden Relics maintains an ongoing commitment to security through regular assessments, secure development practices, and prompt remediation of identified issues. We use encrypted communications (TLS), secure password storage, multi-factor authentication support, and follow OWASP guidelines for web application security.

If you have any security concerns or wish to report a vulnerability, please contact us at edenrelics@dcp-net.com.
